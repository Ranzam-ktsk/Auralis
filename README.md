# Auralis

## 日本語 / Japanese

モダンUI志向の  **音楽管理・再生** と **yt-dlp機能の統合** を実現したデスクトップアプリです。


単なるGUIフロントエンドではなく、  **Windows 11** や **Apple製品** のようなモダン感と操作性を重視して開発されています。



## 目的

yt-dlpは非常に高機能ですが、

- コマンド操作が必要
- オプション設定が複雑
- ダウンロード管理が煩雑
- 音楽プレイヤーとの統合性が低い

といった点から、  一般ユーザーには扱いづらい部分があります。

Auralisは、それらをより直感的に扱えるようにし、

- 音楽の管理
- ダウンロード（ベータ版）
- 再生
- ライブラリ整理

を、1つのアプリで快適に行えるよう設計されています。


## 特徴

- **Windows 11 メディアプレイヤー風** のローカル音楽管理
- **yt-dlp のGUI統合**
- **Windows 11 / Apple風モダンUI**
- 透明感やアニメーションを重視したデザイン
- **プレイリスト管理機能**
- **再生ビジュアライザー**
- **PiP（ピクチャーインピクチャー）対応**
- スマホ音楽アプリ風の縦長プレイヤーUI
- キャッシュ・履歴の個別管理
- 実験的機能を追加可能な **ラボ機能**



## 主な機能

### 音楽ライブラリ

- 設定した音楽ライブラリフォルダーを自動読み込み
- アルバム単位で表示
- ジャンル別表示に対応
- アーティスト / アルバム / 曲名ごとの個別ページ表示
- アーティスト画像を任意で設定可能



### アーティスト管理

複数アーティスト名が

`、` または `,`

で区切られている場合、自動で分離されます。

例：

`A, b`

↓

- A
- B

分離後は、  
それぞれのアーティストページで関連する曲・アルバムを表示します。



### ホーム / 検索

- ホームにおすすめ楽曲を表示
- 「もう一度聞く」から最近再生した曲を表示
- 検索ページから以下を検索可能
  - アーティスト名
  - アルバム名
  - 曲名



### プレイリスト

- プレイリストの作成・編集・再生
- Windows 11 メディアプレイヤーの  
  `.m3u8` プレイリストをそのまま利用可能
- プレイリスト保存先を指定可能

曲追加時は、  
選択した状態を維持したまま別ページへ移動可能です。

そのため、

- 複数アルバム
- 複数ページ

を跨いで曲を選択できます。

※ 現在、アルバム追加は単体対応です。
今後、さらに柔軟な組み合わせ追加へ対応予定です。



### お気に入り

- 曲のお気に入り追加
- アルバム単位でのお気に入り追加
- 後で聞く用途として利用可能
- プレイリスト同様、選択状態維持に対応



### プレイヤー機能

- 再生キュー機能
- 再生キュー自動表示の ON / OFF
- 再生コントロールパネル自動非表示の ON / OFF
- 再生コントロールパネルから音量変更可能
- アプリ全体のマスター音量設定に対応



### 再生ビジュアライザー

- 再生ビジュアライザー表示
- 音域ごとの強度調整に対応



### ピクチャーインピクチャー

- 再生コントロールパネルから PiP 表示可能
- スマホ音楽アプリ風の縦長UI
- ビジュアライザー表示 / 非表示切り替え
- 昔の音楽プレイヤー風デザインを再現



### キャッシュ / データ管理

- キャッシュ管理機能
- 履歴管理機能
- キャッシュと履歴を個別に削除可能
- アルバム画像キャッシュ対応



## ラボ機能（実験的機能）

### yt-dlp統合

設定の **「ラボ機能」** から  
**YoutubeMusic連携** をONにし、

**「yt-dlpをダウンロード / 更新」**

を実行後、  
ダウンロード先フォルダーを指定することで利用できます。


### 利用方法

1. YoutubeMusicタブから曲またはアルバムを開く
2. **「ダウンロードに追加」** をクリック
3. 右サイドバーのダウンロードリストへ追加
4. **「一括ダウンロード開始」** を押すことでダウンロード開始

本アプリは、

- UI
